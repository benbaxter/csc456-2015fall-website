The Voting Game

Summary

You and your group will be building a card game and making it an online card game that you can play with your friends. You will be able to create games, invite friends, track rankings with friends, and more!
    Requirements Grading Criteria Project Implmentation Presentation

Requirements

  1. As a user, I should be able to create a game.
    1. As a user, I should be able to create new question cards that will ask any question the user inputs for that card. (Imagine having blank cards that you can write your questions on)
  2. As a user, I should be able to invite unique people to a game I created. I can send invites via email and facebook.
    1. (For extra credit, let the user set preferences for how they wish to be notified for being invited email vs. facebook vs. desktop notification vs. what ever other notifications you allow)
  3. As a user, I should be able to join a game.
  4. As a system, I will allow a game to be started once 5 or more unique people join.
  5. As a user, if I created a game, I should be notified once 5 people have joined the game.
  6. As a user, if I created a game, I should be able to start the game once it has enough players.
  7. As a user, once a game is started, I should not be able to join.
  8. As a system, I should randomly pick a question to show the players.
    1. This question must be a unique question to this game. Aka no duplicates questions may be asked during one game
  9. As a user, I should be able to vote for anthoer player as my answer to the question.
  10. As a system, I should collect all of the answers and once I have them all, I should show the players who had the most votes. (Note: What about ties? The player with the most votes wins this round.)
  11. As a system, I should repeat asking questions (dealing question cards) and collecting answers until one player has won 5 times.
  12. As a system, once somoeone has won 5 times, I should notify all the players who the winner is.
  13. As a system, I should archive the game and the statistics for all of the players in the game so that they may go back and see how many games they won/lost.
  14. As a user, I should be able to login and see my ranking and compare how many games I won with my friends. (Think leaderboards)
  15. As a system, I should provide a way for an admin user to upload a bulk set of questions that would be used for all the games.
  16. As a user, I should be able to register and create an account.
  17. As a user, I should be able to sign in with Facebook, Google, or any other social network by using oauth
